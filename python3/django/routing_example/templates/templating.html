This is an HTML template.

It can accept data like this:

Data: {{ cool_data }}

It also has support for lists or anything that can be iterated over:

    {% for item in cool_list %}
  • {{ item }}
    • {% endfor %}

As well as dictionaries or things with properties:

    {% for key, value in cool_dict.items %}
  • {{ key }}: {{ value }}
    • {% endfor %}
