<b>SYSTEM REQUIREMENTS:</b><br>
This is a list of system requirements that the games can have. Each game has its own listing of sys reqs. Some of them are both for online and download versions, while some are only for the download versions. PLEASE NOTE: the sys req listings may not be correct, please check them too.<br>
<br>
OS reqs:<br>
Windows<br>
Windows/Mac<br>
<br>
Browser reqs:<br>
Firefox recommended<br>
modern browser<br>
<br>
Plugin reqs:<br>
Flash<br>
Flash (trusted folder settings change required to play in a browser)<br>
Shockwave<br>
Shockwave (Xtras patch recommended -> Windows recommended for auto Xtras patch)<br>
Shockwave 8 (old browser required -> Windows recommended to install browser pre-configured to use S8)<br>
Shockwave 8 (Xtras patch recommended -> Windows recommended for auto Xtras patch)<br>
Unity3D<br>
Cult3D and old Java (old browser required -> Windows recommended to install browser pre-configured to use C3D and old J)<br>
Java (to play on FireFox, installation to a local server program [Windows recommended for XAMPP and auto-installation to it])<br>
<br>
Other reqs:<br>
installation to local server program (Windows recommended for XAMPP and auto-installation to it)<br>
proxy redirect (Windows recommended for automatic proxy settings change)<br>
proxy redirect (Windows recommended to install browser pre-configured to use the needed proxy redirect)<br>
none<br>
